
# Wazuh SIEM Brute Force Detection Lab

## Overview

This project demonstrates brute-force attack detection using Wazuh SIEM in a VMware home lab.

## Technologies

- Wazuh SIEM
- Windows Server 2022
- Ubuntu
- VMware
- Wireshark

## Project

A brute-force attack was simulated against a Windows Server. Wazuh successfully detected the attack and generated security alerts. The generated logs were analyzed to verify the detection process.
